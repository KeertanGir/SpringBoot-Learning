# 📖 About

This repository follows a **Monorepo** approach and contains multiple independent projects developed while learning Java and Spring Boot.

The purpose of this repository is to experiment with different backend concepts such as:

- REST APIs
- Spring MVC
- Spring Data JPA
- Hibernate
- Validation
- HATEOAS
- Security Concepts
- CRUD Applications
- Backend Architecture Patterns
